## 3. 機能ドキュメントの主要な変更

### Code Review (`code-review.md`)
- **Manual モード追加**: レビュートリガーに「Manual」を新規追加（`@claude review` コメントでのみ開始）
- **`@claude review` セクション新設**: PR コメントで手動レビュー開始。トップレベルコメント・先頭記述・owner/member/collaborator 権限が必要
- レビュートリガー名を刷新: 「After PR creation only」→「Once after PR creation」、ラベルを「Review Behavior」に変更

### Remote Control (`remote-control.md`)
- **サーバーモードとインタラクティブモードの区別を導入**:
  - `claude remote-control` = サーバーモード（ローカル UI なし）
  - `claude --remote-control` / `--rc` = インタラクティブセッション + リモート制御
- **`--spawn` フラグ追加**: `same-dir`（デフォルト）と `worktree`（各セッションに git worktree 割当）
- **`--capacity` フラグ追加**: 同時セッション上限（デフォルト32）

### MCP (`mcp.md`)
- **MCP elicitation セクション新設**: フォームモード（構造化フィールド）と URL モード（ブラウザ認証）の説明

### モデル設定 (`model-config.md`)
- **1M コンテキストウィンドウ対応に Opus 4.6 追加**。「beta」注記を削除
- プラン別利用可否テーブル追加（Max/Team/Enterprise: 含む、Pro: エクストラ使用量）
- `opus[1m]` エイリアス追加
- **effort レベルに `max` 追加**（Opus 4.6 のみ、セッション限定、トークン制限なし）
- `/effort` コマンドと `--effort` フラグによる設定方法を追加

### Adaptive Reasoning (`common-workflows.md`)
- 対応モデルを Opus 4.6 のみから **Opus 4.6 + Sonnet 4.6** に拡大

### Hooks (`hooks.md`, `hooks-guide.md`)
- **`/hooks` メニューが読み取り専用に変更**: フック追加・削除は設定 JSON 直接編集に統一
- `PostCompact`、`Elicitation`、`ElicitationResult` フックの完全なリファレンスを追加
- ライフサイクル図を新イベント含む版に差し替え

### Settings (`settings.md`)
- `effortLevel`、`feedbackSurveyRate` 設定キー追加
- 「Worktree settings」セクション新設（`worktree.symlinkDirectories`、`worktree.sparsePaths`）
- Windows レガシーパス廃止の Warning 追加

### Google Vertex AI (`google-vertex-ai.md`)
- 1M コンテキスト対応モデルを拡大: Opus 4.6、Sonnet 4.6、Sonnet 4.5、Sonnet 4 に

### Memory (`memory.md`)
- CLAUDE.md の配信方法を明確化: 「システムプロンプトの一部ではなく、その後のユーザーメッセージとして配信」

### Network Config (`network-config.md`)
- GitHub Enterprise Cloud の IP 許可リスト設定手順を追加

## 4. CLI リファレンスの変更

### 新規フラグ
| フラグ | 説明 |
|---|---|
| `--effort` | effort レベル設定（low/medium/high/max） |
| `--name` / `-n` | セッション表示名設定 |
| `--remote-control` / `--rc` | インタラクティブセッションでリモート制御を有効化 |

### 変更されたフラグ
| フラグ | 変更内容 |
|---|---|
| `--plugin-dir` | 複数パス → 1パスのみ（複数はフラグ繰り返し） |

### 新規コマンド
| コマンド | 説明 |
|---|---|
| `/effort` | effort レベル設定（low/medium/high/max/auto） |
| `/feedback` | フィードバック送信（`/bug` のエイリアス） |
| `/rename` | セッション名変更 + プロンプトバー表示 |
