- **`/resume` のセッションピッカー:** `/resume` が削除済みセッションを含む過去セッションのピッカーを開き、バックグラウンドセッションとして再開できるように。

**主なバグ修正:**

- `/ultrareview` が PR 参照を拒否していた問題を修正
- セッション再開時に「File has not been read yet」エラーが出る問題を修正
- 画像が多い会話で「Request too large」になる問題を修正

**改善:**

- LLM ゲートウェイ・カスタム base URL 経由でもプロンプトキャッシュが機能するように
- バックグラウンドエージェントへのアタッチ時に即座にフォーマット済みトランスクリプトを表示

### 1. Artifacts：MCP コネクターから Live データ取得（新機能）

**URL:** https://code.claude.com/docs/en/artifacts.md

**変更内容:**

アーティファクトが MCP コネクターを呼び出してリアルタイムデータを取得できるようになりました（v2.1.209 以降）。

- 公開されたアーティファクトが閲覧時に MCP コネクター経由でライブデータを取得可能
- ダッシュボードが「セッション構築時のスナップショット」ではなく常に最新データを表示
- **公開リンクへの共有不可**（コネクター対応アーティファクトはすべてのプランで非対応）
- チームと Enterprise プランでは組織内共有が可能
- 閲覧者は自分のアカウントのコネクター経由で呼び出しが行われる（発行者のアカウントは不使用）
- 閲覧者ごとに最初のコネクター呼び出し前に許可確認が発生

**新設セクション:** "Pull live data with MCP connectors"、"How connector calls work for viewers"

**また同時に追加:**
- 公開共有リンク機能（Pro・Max プランでの公開リンク）
