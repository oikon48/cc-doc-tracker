## 1. CHANGELOG ページの更新

> **URL**: https://code.claude.com/docs/en/changelog.md

**v2.1.207**（2026年7月11日リリース）が新たに追加されました。

### 主な変更内容

| カテゴリ | 変更内容 |
|---|---|
| **Auto mode** | Bedrock・Vertex AI・Foundry でも `CLAUDE_CODE_ENABLE_AUTO_MODE` 環境変数なしで Auto mode が使用可能に。無効化は設定の `disableAutoMode` で行う |
| **デフォルトモデル変更** | Bedrock・Vertex・Claude Platform on AWS のデフォルトが Claude Opus 4.8 に変更 |
| **Auto mode 設定読み込み先変更** | `autoMode` 設定をリポジトリ内の `.claude/settings.local.json` から読まなくなった。`~/.claude/settings.json`（ユーザー設定）を使用すること |
| **プラグインセキュリティ強化** | プラグインの shell-form コマンドで `${user_config.*}` が使用不可に（シェルインジェクション対策）。exec form（`args` 配列）または `$CLAUDE_PLUGIN_OPTION_<KEY>` を使用する |
| **プラグイン設定の読み込み制限** | `pluginConfigs`（プラグインオプション値）をプロジェクトレベルの `.claude/settings.json` から読まなくなった |
| **不具合修正（多数）** | ターミナルのフリーズ・遅延（長い一覧やコードブロック受信時）、偽の prompt-injection 警告、auto-updater が `~/.local/bin/claude` のカスタムランチャーを上書きする問題、Windows での AWS 認証情報解決ハング、`/usage-credits` の入力バリデーション強化など |
| **エージェントビュー改善** | テキスト貼り付け時の重複プレースホルダー解消、ブロックセッションの質問表示改善 |
