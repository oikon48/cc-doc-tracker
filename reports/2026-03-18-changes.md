## 新規ドキュメント

### voice-dictation.md（音声ディクテーション）

Push-to-talk による音声入力機能の完全なドキュメントが新設された。

- `Space` キー長押しで録音開始、離すと文字起こしがプロンプトに挿入
- `/voice` コマンドで有効化/無効化を切り替え
- Claude.ai アカウントでの認証が必須（API キーや Bedrock/Vertex では不可）
- ローカルマイク必須のためリモート環境（Web版、SSH）では使用不可
- 20言語対応（日本語含む）、`language` 設定と連動
- `keybindings.json` で `voice:pushToTalk` アクションのキーバインド変更可能
- トラブルシューティングガイド付き

## v2.1.77 リリースノート (changelog.md)

### 主要な新機能・改善

- Opus 4.6 のデフォルト最大出力トークン数を 64k に引き上げ、上限は Opus/Sonnet 4.6 共に 128k
- `allowRead` サンドボックス設定を追加（`denyRead` 領域内で特定パスの読み取りを再許可）
- `/copy N` で N 番目に最新のアシスタント応答をコピー可能に
- `/fork` → `/branch` にリネーム（`/fork` はエイリアスとして残存）
- プラン承認時にセッション自動命名
- `claude plugin validate` でスキル/エージェント/フックのフロントマターと `hooks/hooks.json` もチェック
- Agent ツールの `resume` パラメータ廃止 → `SendMessage({to: agentId})` に変更
- `SendMessage` が停止済みエージェントを自動再開
- macOS での起動高速化（~60ms）
- `--resume` の高速化（最大45%、メモリ100-150MB削減）

### 主要なバグ修正

- 複合 Bash コマンドの「Always Allow」でサブコマンドごとにルール保存するよう修正
- 自動アップデーターのメモリリーク修正（数十GBメモリ消費）
- `--resume` の会話履歴切り詰めレースコンディション修正
- PreToolUse フックの `"allow"` が `deny` パーミッションルールをバイパスしていた問題を修正
- Write ツールが CRLF ファイルの改行を変換していた問題を修正
- Vim NORMAL モードで Backspace/Delete が動作しない問題修正
- tmux 内でのクリップボードコピー、背景色レンダリング等の修正
- バックグラウンド Bash タスクが出力 5GB 超過で自動終了
